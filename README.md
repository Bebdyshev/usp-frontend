# Единый профиль учащигося

## Installation

Frontend:

```bash
npm i
npm run dev
```
Backend:
[Backend repository](https://github.com/Bebdyshev/usp-backend)
```bash
pip install -r requirements.txt
uvicorn app:app --reload
```
